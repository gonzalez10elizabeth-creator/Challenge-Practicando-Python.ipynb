# Challenge-Practicando-Python.ipynb
El objetivo de este proyecto es analizar el desempeño de cuatro tiendas de Alura Store, identificando cuál de ellas presenta menor rendimiento y podría considerarse para la venta.
Se evaluaron indicadores clave como:

Facturación total de cada tienda

Categorías de productos más y menos vendidas

Calificación promedio de los clientes

Productos más y menos vendidos

Costo de envío promedio

A partir de estos análisis, se generaron visualizaciones para facilitar la interpretación de los datos y respaldar la toma de decisiones.

Estructura del proyecto
